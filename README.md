# 💰 Personal Finance & Expense Management System (Excel Project)

## 🎯 Objective
To track income, expenses, and savings while analyzing financial health using an interactive Excel dashboard.

---

## 🛠 Tools Used
- Microsoft Excel  
- Pivot Tables  
- Charts & Dashboards  
- Basic Financial Formulas  

---

## 📂 Dataset Overview
Contains monthly financial data including:
- Income sources  
- Expense categories  
- Payment modes  
- Budget vs actual spending  
- Savings tracking  

---

## 📊 Key Features
- Income vs Expense tracking  
- Budget monitoring  
- Savings analysis  
- Category-wise expense breakdown  
- Interactive dashboard with filters  

---

## 📈 Key KPIs
- Total Income  
- Total Expenses  
- Net Savings  
- Savings Rate %  
- Budget vs Actual Spend  

---

## 📊 Dashboard Insights
- Identifies major expense areas  
- Tracks savings trends over time  
- Highlights overspending categories  
- Provides overall financial health view
